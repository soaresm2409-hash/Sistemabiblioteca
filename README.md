# Sistemabiblioteca
Sistema de gerenciamento de livros desenvolvido em C
Este é um projeto acadêmico desenvolvido para praticar os fundamentos da linguagem C e lógica de programação.
- Cadastro de livros (ID, Título, Autor, Ano).
- Persistência de dados: os livros são salvos em um arquivo `.txt`.
- Menu interativo com `switch/case`.
- Validação básica de dados.
COMO RODAR O PROJETO.
1. Tenha um compilador de C instalado (GCC).
2. Compile com: `gcc biblioteca.c -o sistema`
3. Execute com: `./sistema`
